# Pakistan Tourism Data Analysis

Exploratory data analysis of Pakistan tourism statistics — visitor trends, popular destinations, and revenue data.

## Contents
- Jupyter notebooks with EDA
- Visualizations (charts, maps)
- Dataset sources and references

## Requirements
```
pip install pandas numpy matplotlib seaborn jupyter
jupyter notebook
```

## Key Findings
- Year-over-year visitor trend analysis
- Top tourist destinations by province
- Revenue and economic impact visualization
